## 📖 О проекте
Данный репозиторий содержит исходный код и документацию выпускной квалификационной работы на тему:  
**«Разработка информационной системы парсинга сайтов для улучшения маркетинговой политики для предприятия "Garage", г. Усть-Лабинск»**.

Система автоматизирует сбор, очистку и структурирование данных с веб-ресурсов конкурентов в сфере автотоваров. Полученная информация используется для сравнительного анализа цен, мониторинга ассортимента и принятия обоснованных маркетинговых решений, что позволяет оптимизировать ценовую политику предприятия.

## ✨ Ключевые возможности
- ✅ **Мультипарсинг:** поддержка 7 крупных интернет-магазинов автозапчастей
- ✅ **Извлечение данных:** название товара, актуальная цена, бренд/производитель, прямая ссылка
- ✅ **Обход динамического контента:** автоматическая прокрутка, нажатие кнопок «Показать ещё», рендеринг JS
- ✅ **Стеалс-режим:** скрытие признаков автоматизации (`webdriver`, `blink features`, `excludeSwitches`)
- ✅ **Удобный CLI:** интерактивное консольное меню для выбора целевого сайта
- ✅ **Экспорт в Excel:** автоматическое формирование структурированных таблиц через `pandas` + `openpyxl`
- ✅ **Масштабируемая архитектура:** динамическая загрузка модулей парсеров через `importlib`
